Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML is the basis for most websites, and some of the most basic coding skills to master.
  • It is important to understand that the only visible part of HTML code is the BODY element.
  • Make sure to save often and stay relaxed. If something is broken, it can always be fixed.

CSS

  • CSS stands for Cascading Style Sheets.
  • CSS describes how HTML elements are to be displayed on screen, paper, or in other media.
  • CSS saves a lot of work. It can control the layout of multiple web pages all at once.
  • External stylesheets are stored in CSS files.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • Git is the basis of "communication" for coding on your personal PC to a data base.
  • The “index”, “staging area” and “cached” are all the same thing.
  • Not all references are branches or tags.
  • If need a reminder of commands, check OverFlow, Reddit, or Google.

JavaScript

  • JavaScript is a powerful programming language that can add interactivity to a website.
  • JavaScript code alters HTML and CSS code to make a website interactive rather than a static page.
  • In other words, "makes it work"
  • Can create things like a pull-down menu and embedded media files.
